# 越海选品 (YueXuan) — Chrome Web Store Listing

## 中文描述 (Chinese Description)

### 标题
越海选品 — 三源数据交叉分析，发现越南蓝海品类

### 简短描述 (132字符以内)
中国海关数据 + 1688出厂价 + Shopee越南行情，三源交叉分析。一键发现越南蓝海品类，精准测算跨境利润。跨境电商卖家必备选品工具。

### 详细描述

**越海选品** 是专为跨境电商卖家打造的中国→越南市场选品分析工具。通过三源数据交叉分析，帮你比市场快 3-6 个月发现蓝海品类。

**核心功能：**

📊 **海关出口趋势追踪**
- 实时查询中国→越南海关出口数据
- 同比增速可视化，暴增品类自动高亮
- 主要出口省份分布，掌握货源地图

🏭 **1688 智能比价**
- 浏览1688商品时自动注入侧边栏
- 一键获取出厂价区间和中位数
- 自动对比Shopee售价，算出价差

🛒 **Shopee 越南市场洞察**
- 浏览Shopee VN时自动叠加海关数据
- 竞争度分析：卖家数、需求趋势、新卖家占比
- 1688类似款价格实时对比

📝 **一键生成分析报告**
- 聚合三源数据，生成结构化Markdown报告
- 利润测算：采购成本 → 运费关税 → Shopee售价 → 毛利率
- 支持复制和下载.md文件

**为什么选越海选品？**
- 数据驱动决策，告别"凭感觉"选品
- 纯浏览器插件，无需安装额外软件
- 深色专业UI，数据一目了然
- 支持Chrome/Edge/360/QQ/搜狗等主流浏览器

**适用人群：**
- 跨境电商卖家（尤其是做越南市场的）
- 外贸SOHO、选品经理
- 供应链公司、采购代理

## English Description

### Title
YueXuan — Cross-Border Vietnam Market Product Sourcing Tool

### Short Description (132 chars max)
Cross-reference China customs export data, 1688 factory prices & Shopee Vietnam listings. Discover blue-ocean product categories before competitors.

### Detailed Description

**YueXuan** is a product sourcing analysis tool for cross-border e-commerce sellers targeting the China→Vietnam market. By cross-referencing three data sources, it helps you discover blue-ocean categories 3-6 months ahead of the market.

**Core Features:**

📊 **Customs Export Trend Tracking**
- Real-time China→Vietnam customs export data queries
- YoY growth visualization with automatic opportunity highlighting
- Top exporting provinces breakdown

🏭 **1688 Price Comparison**
- Auto-injected sidebar when browsing 1688 product pages
- Factory price range and median extraction
- Automatic price gap calculation against Shopee listings

🛒 **Shopee Vietnam Market Insights**
- Customs data overlay when browsing Shopee VN
- Competition analysis: seller count, demand trends, new seller ratio
- Real-time 1688 similar product price comparison

📝 **One-Click Report Generation**
- Aggregated structured Markdown reports from all three data sources
- Profit calculation: sourcing cost → shipping & tariff → Shopee price → gross margin
- Copy to clipboard and .md file download

**Why YueXuan?**
- Data-driven product sourcing decisions
- Pure browser extension — no additional software needed
- Professional dark-themed UI
- Compatible with Chrome, Edge, and major Chinese browsers

**Ideal for:**
- Cross-border e-commerce sellers (especially Vietnam market)
- Product sourcing managers and trade SOHOs
- Supply chain companies and procurement agents
